# Mean_Filter:The program explaining the usage of the mean filter. The filter (k) is an ndarray array of size 5-by-5 with all values = 1/25. The filter is then convolved using the “convolve” function from scipy.ndimage.filters.

--Advantages of the mean filter:

•Removes noise.

•Enhances the overall quality of the image, i.e., mean filter brightens an image.

--Disadvantages of the mean filter:

•In the process of smoothing, the edges get blurred.

•Reduces the spatial resolution of the image.
